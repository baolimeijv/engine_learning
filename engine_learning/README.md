# 引擎学习笔记

本项目记录我学习游戏引擎开发的过程，包含手写底层代码和原理笔记。

## 已完成模块

- [内存池笔记](./memory_pool.md) — 定长内存池实现与原理
- [智能指针笔记](./shared_ptr.md) — 简化版 shared_ptr 实现与引用计数原理
- [数学库笔记](./matrix4.md) — 手写 Vector3、Matrix4，支持 MVP 变换、透视投影
- [软渲染器笔记](./soft_renderer.md) — 手写光栅化、深度缓冲、背面剔除、Flat Shading 光照（环境光+漫反射+高光）

## 技术栈

- C++17 / C++11
- 手动内存管理、虚函数表、内存池、智能指针
- 图形学基础：MVP 变换、透视投影、光栅化、深度缓冲、背面剔除、Blinn-Phong 光照
- 第三方库：SFML 3.0.2（用于窗口显示和像素操作）
