## 3. NDC 到屏幕的 y 轴反向

### 结论
NDC 到屏幕的 y 映射是关于水平轴（x 轴）的轴对称变换（上下翻转）
屏幕左上角为原点（y 向下），NDC 左下角为原点（y 向上）。

## 4. 透视投影后 z_ndc 与 Zc 的单调性

### 结论
- 相机空间：近处 Zc 大（如 -5），远处 Zc 小（如 -50）。
- NDC：近处 z_ndc = -1，远处 z_ndc = 1。
- z_ndc 随 Zc 增大而减小（单调递减），即“单调性反转”。

深度测试使用 z_ndc < buffer 仍正确保留近处物体。

## 6. 环境光 (Ambient)
- 模拟间接光照，使背光面不至于全黑。
- 公式：`I_ambient = ka * Ia`  
  - `ka`：环境光反射系数（通常 0.1～0.3）  
  - `Ia`：环境光强度（通常固定为 0.2 左右）
- 在你的代码中：`ambient = 0.2f`，直接加到总强度中。

## 7. 漫反射 (Diffuse)
### 原理
- 朗伯余弦定律：单位面积接收的光能量与光线方向 `L` 和法线 `N` 的余弦成正比。
- 公式：`diff = kd * max(0, N · L)`
  - `kd`：漫反射系数（材质的颜色）
  - `L`：从表面指向光源的单位向量
  - `N`：表面单位法向量

### 代码实现
```cpp
vector3 L = lightDir.normalized();
float diff = std::max(0.0f, normal.dot(L));
float intensity = ambient + diff;
